# Tools — Vault Araçları

Bu dizin, vault üzerinde çalışan CLI araçlarını içerir.

## Araçlar

### Arama Motoru
> `scripts/search.py` — Vault üzerinde tam metin arama

Küçük, naif bir arama motoru. WIKI makalelerini indeksler ve sorgu sonuçlarını döndürür.
LLM ajanı tarafından araç olarak kullanılabilir.

### Diğer araçlar

| Araç | Açıklama | Durum |
|------|----------|-------|
| Arama motoru | Wiki üzerinde tam metin arama | Planlandı |
| Backlink doğrulayıcı | Kopuk bağlantıları kontrol et | Planlandı |
| İstatistik toplayıcı | Wiki metriklerini hesapla | Planlandı |

## Not

`search_index/` altındaki dosyalar otomatik oluşturulur ve sürüm kontrolüne dahil edilmeyebilir.
